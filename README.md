Vyzorix is an AI-powered software development platform. It provides agents that can perform development tasks by interacting with tools, browsing the web, and modifying code.

There are a few ways to work with Vyzorix:

### Software Agent SDK
The SDK is a composable Python library that contains all of the agentic tech. It's the engine that powers everything else below.

Define agents in code, then run them locally, or scale to 1000s of agents in the cloud.

### CLI
The CLI is the easiest way to start using Vyzorix. The experience will be familiar to anyone who has worked
with e.g. Claude Code or Codex. You can power it with Claude, GPT, Gemini, or any other LLM.

### Local GUI
Use the Local GUI for running agents on your laptop. It comes with a REST API and a single-page React application.

### Enterprise
The enterprise directory contains source-available features and integrations:
- Multi-user support
- RBAC and permissions
- Collaboration features (e.g., conversation sharing)
- Integrations with Slack, Jira, and Linear
